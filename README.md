# get-design CLI

**A CLI for get-design.**
**Fetch design tokens you need from webpages via CLI!**

# Pre-requisites

- Install [Node.js](https://nodejs.org/en/)

# Installation

clone & use /bin/get-design

#### Expect npm package soon!

# Usage

Get background-colors:

```bash
$ get-design bgc [URL]
```
